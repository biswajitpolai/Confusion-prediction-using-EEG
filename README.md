# Confusion-prediction-using-EEG
Using EEG signal from adults watching MOOC video  clips in a pilot study, i trained and tested classifiers to detect when the student  is confused while watching the course material.
#citation
Wang, H., Li, Y., Hu, X., Yang, Y., Meng, Z. and Chang, K.M. (2013) Using EEG to Improve Massive Open Online Courses Feedback Interaction. CEUR Workshop Proceedings, 1009, 59-66.
